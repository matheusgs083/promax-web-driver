# Tests

Testes automatizados organizados por responsabilidade, seguindo o mesmo padrao adotado em `core/` e `pages/`.

Subpastas:

- `core/`
  - contratos e servicos da camada compartilhada
- `pages/`
  - testes dos page objects e classes base de navegacao
- `support/`
  - doubles, fakes e utilitarios de apoio aos testes

Arquivos de apoio na raiz:

- `conftest.py`
  - fixtures compartilhadas
